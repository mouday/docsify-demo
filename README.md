# docsify

docsify 可以快速帮你生成文档网站

文档：https://docsify.js.org/#/zh-cn/

## 快速开始

```bash
# 安装依赖
npm i docsify-cli -g

# 初始化项目
docsify init ./docs

# 本地预览
docsify serve docs

```
